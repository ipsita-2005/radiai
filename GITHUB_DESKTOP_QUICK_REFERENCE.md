## Daily Workflow (3 Steps)

### 1️⃣ Make Changes
```
Edit files → Save in VS Code
↓
Changes appear in GitHub Desktop "Changes" tab
```

### 2️⃣ Commit
```
Write summary (what changed)
Click "Commit to main"
```

### 3️⃣ Push
```
Click "Push origin" (top right)
✅ Synced to GitHub!
```

## If Something Goes Wrong

### "Authentication Failed"
1. Go to GitHub.com → Settings → Developer settings → Personal access tokens
2. Generate new token (classic) with repo scope
3. Use token as password when prompted

### "Merge Conflicts"
1. GitHub Desktop shows conflicts
2. Click each conflicted file
3. Choose which changes to keep
4. Mark as resolved → Commit merge

### "Repository Not Found"
1. File → Add Local Repository
2. Browse to ashwini folder
3. Click "Add Repository"
