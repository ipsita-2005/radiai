# 🚀 Quick Start: Deploy with Gemini AI

## ⚡ 3-Minute Setup

### Step 1: Get FREE API Key (1 minute)

**Go to:** https://aistudio.google.com/app/apikey

1. Sign in with Google
2. Click "Create API Key"
3. Copy the key

**Done!** ✅

---

### Step 2: Add API Key to Vercel (1 minute)

**Go to:** https://vercel.com/dashboard/ipsita-2005/radiai/settings/environment-variables

1. Click "Add Environment Variable"
2. Name: `VITE_GEMINI_API_KEY`
3. Value: Paste your API key
4. Click "Save"

**Done!** ✅

---

### Step 3: Redeploy (2 minutes)

**On Vercel Dashboard:**

1. Go to "Deployments" tab
2. Click "Redeploy" on latest deployment
3. Wait 2-3 minutes
4. Click "Visit" when ready

**Done!** ✅

---

## 🔧 Troubleshooting

### "API key not configured" error?

**Fix:**
1. Check Vercel → Settings → Environment Variables
2. Ensure `VITE_GEMINI_API_KEY` is set
3. Redeploy

### Want to test without API key?

Just leave `VITE_GEMINI_API_KEY` empty - app falls back to demo mode automatically!
